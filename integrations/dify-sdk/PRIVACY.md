## Privacy Policy

### Data Collection

This plugin collects and transmits the following data to your self-hosted Cognee server:

- **Login credentials** (email, password) - used solely for authenticating requests to your Cognee instance.
- **Text data** - content you explicitly provide through the Add Data tool, sent to your Cognee server for ingestion.
- **Search queries** - queries you provide through the Search tool, sent to your Cognee server for processing.
- **Dataset and data identifiers** - UUIDs provided for delete operations.

### Data Usage

- All data is transmitted directly to your own self-hosted Cognee server instance. This plugin acts as a pass-through and does not store, log, or process any data independently.
- Login credentials are stored securely by Dify and are only used to authenticate requests to your Cognee instance.

### Third-Party Services

This plugin communicates with the Cognee server at the base URL you configure. Since you control the server, data stays within your infrastructure. Please refer to [Cognee's privacy policy](https://www.cognee.ai/privacy) for details on how the Cognee software handles your data.

### Data Sharing

This plugin does not share any data with third parties. All communication is between the Dify instance and your self-hosted Cognee server.
