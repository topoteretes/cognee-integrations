## Privacy Policy

### Data Collection

This plugin collects and transmits the following data to the Cognee API:

- **API credentials** (Base URL, API Key) - used solely for authenticating requests to your Cognee instance.
- **Text data** - content you explicitly provide through the Add Data tool, sent to your Cognee API for ingestion.
- **Search queries** - queries you provide through the Search tool, sent to your Cognee API for processing.
- **Dataset and data identifiers** - UUIDs provided for delete operations.

### Data Usage

- All data is transmitted directly to your own Cognee API instance. This plugin acts as a pass-through and does not store, log, or process any data independently.
- API credentials are stored securely by Dify and are only used to authenticate requests to your Cognee instance.

### Third-Party Services

This plugin communicates with the Cognee API at the base URL you configure. Please refer to [Cognee's privacy policy](https://www.cognee.ai/privacy) for details on how your data is handled by the Cognee service.

### Data Sharing

This plugin does not share any data with third parties beyond your configured Cognee API instance.
